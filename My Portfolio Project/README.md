# محفظة QTBM - ملف المشروع الكامل (Full Portfolio Bundle)

يحتوي هذا المجلد على النسخة الكاملة والمصقولة من تطبيق **QTBM Wallet**. تم تنظيم الملفات لتعكس الهيكل البرمجي الحقيقي للتطبيق (Next.js 15 + Firebase + Genkit).

## محتويات المجلد:
1. **src/app**: كافة صفحات التطبيق (لوحة التحكم، الخدمات، الإدارة، الملف الشخصي).
2. **src/components**: المكونات الرسومية، الشعار المتحرك، ونوافذ الأوامر الصوتية.
3. **src/ai**: تدفقات الذاء الاصطناعي للأوامر الصوتية.
4. **التهيئة**: ملفات `backend.json` و `firestore.rules` و `firebase-config.ts`.

## المميزات التقنية المضمنة:
- **واجهات Glassmorphism**: تصميم عصري يعتمد على الشفافية والظلال العميقة.
- **دعم العملات المتعددة**: نظام إدارة أرصدة (YER, USD, SAR).
- **أوامر صوتية ذكية**: معالجة اللغة الطبيعية لتنفيذ التحويلات.
- **لوحة تحكم إدارية**: لإدارة المنتجات والموافقات المالية.
