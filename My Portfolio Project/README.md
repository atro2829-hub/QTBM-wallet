# محفظة QTBM - ملف تكوين المشروع (Project Portfolio)

يحتوي هذا المجلد على كافة المفاتيح البرمجية، قواعد الأمان، ومخطط البيانات الخاص بتطبيق **QTBM Wallet**. تم تنظيم هذه الملفات لتسهيل عملية الرفع والتصدير كجزء من ملفك الشخصي البرمجي.

## محتويات المجلد:
1. **firebase-config.ts**: يحتوي على بيانات الاتصال بـ Firebase (API Keys, Project IDs).
2. **backend.json**: المخطط الهيكلي لقاعدة البيانات (Entities & Paths).
3. **firestore.rules**: قواعد الأمان الصارمة التي تحمي بيانات المستخدمين.
4. **.env**: ملف متغيرات البيئة.

## عن المشروع:
تطبيق محفظة رقمية متطور مبني باستخدام Next.js 15 و Firebase، يدعم العملات المتعددة (YER, USD, SAR) ويتميز بنظام توثيق هوية صارم وواجهات Glassmorphism عصرية.

تم إعداد هذا المجلد بواسطة فريق تطوير QTBM.
