# محفظة QTBM - دليل بناء APK النهائي

لضمان نجاح بناء التطبيق وتجنب أخطاء SDK، يرجى اتباع التعليمات التالية بدقة.

### الخطوة 1: ضبط مسار الـ SDK
إذا ظهر لك خطأ "SDK Location not found"، افتح ملف `android/local.properties` وقم بتعديل سطر `sdk.dir` ليطابق مكان تثبيت الأندرويد على جهازك:

*   **على Linux:** عادة ما يكون `/home/user/Android/Sdk`
*   **على Windows:** عادة ما يكون `C:/Users/اسم_المستخدم/AppData/Local/Android/Sdk` (استخدم السلاش الأمامي `/` أو `\\`).
*   **على Mac:** عادة ما يكون `/Users/اسم_المستخدم/Library/Android/sdk`

### الخطوة 2: تشغيل أوامر البناء
**يجب دائماً تنفيذ الأوامر من المجلد الرئيسي للمشروع (studio وليس studio/android).**

إذا كنت داخل مجلد android حالياً، اخرج منه أولاً:
```bash
cd ..
```

ثم قم بتشغيل أمر البناء الشامل:
```bash
npm run apk:full-build
```
