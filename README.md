# محفظة QTBM - التميز المالي الرقمي (QTBM Wallet)

تطبيق محفظة رقمية متطور مبني باستخدام Next.js 15 و Firebase، مصمم لتوفير تجربة مالية آمنة وسلسة.

## المميزات الرئيسية
- **إدارة العملات المتعددة**: (YER, USD, SAR) بشكل منفصل تماماً.
- **شراء USDT**: نظام شراء متكامل بعمولة 5%.
- **لوحة تحكم إدارية**: إدارة المستخدمين، المنتجات، والتحويلات.
- **أوامر صوتية بالذكاء الاصطناعي**: تنفيذ العمليات عبر الصوت.
- **واجهات عصرية**: تصميم Glassmorphism مع دعم الوضع الليلي.

## دليل تصدير التطبيق كملف APK (أندرويد)

لتحويل هذا المشروع إلى تطبيق أندرويد، اتبع الخطوات التالية:

### 1. المتطلبات الأساسية
- تثبيت **Android Studio** على جهازك.
- تثبيت **Java SDK (JDK 17)**.

### 2. إعداد Capacitor
قم بتشغيل الأوامر التالية في مجلد المشروع:
```bash
# 1. تهيئة Capacitor في المشروع (قم بتغيير اسم الحزمة والاسم كما تراه مناسباً)
npx cap init [app-name] [app-id] --web-dir out

# 2. إضافة منصة الأندرويد
npx cap add android
```

### 3. بناء المشروع وتصديره
في كل مرة تريد فيها تحديث التطبيق، قم بتشغيل:
```bash
# 1. بناء نسخة الويب الثابتة
npm run build

# 2. مزامنة الملفات مع مجلد الأندرويد
npx cap sync

# 3. فتح المشروع في Android Studio لبناء الـ APK
npx cap open android
```

### 4. إنشاء ملف APK النهائي
داخل **Android Studio**:
1. اذهب إلى قائمة **Build**.
2. اختر **Build Bundle(s) / APK(s)**.
3. اختر **Build APK(s)**.
4. سيقوم البرنامج بإنشاء الملف وتزويدك بمسار الوصول إليه.

---
تم التطوير بواسطة **فريق QTBM الرقمي**.
