
# محفظة QTBM - التميز المالي الرقمي (QTBM Wallet)

تطبيق محفظة رقمية متطور مبني باستخدام Next.js 15 و Firebase، مصمم لتوفير تجربة مالية آمنة وسلسة. تم تحسين هذا المشروع ليعمل كتطبيق أندرويد أصلي (APK).

## 🚀 دليل تصدير التطبيق كملف APK

للحصول على مجلد `android` جاهز للاستخدام، اتبع الخطوات التالية في الترمنال (Terminal) بالترتيب:

### 1. تنفيذ الأوامر في Terminal
```bash
# 1. بناء نسخة الويب الثابتة (سيتم إنشاء مجلد out)
npm run build

# 2. مزامنة ملفات البرمجة مع مجلد الأندرويد
npx cap sync
```

### 2. إنشاء ملف APK النهائي
1. افتح برنامج **Android Studio**.
2. اختر **Open Project** وقم باختيار مجلد `android` الموجود داخل هذا المشروع.
3. انتظر انتهاء المزامنة، ثم من القائمة العلوية: **Build** > **Build Bundle(s) / APK(s)** > **Build APK(s)**.
4. ستجد الملف في المجلد الذي يفتحه لك الرابط "locate".

---
تم التطوير بواسطة **فريق QTBM الرقمي**.
