# محفظة QTBM - التميز المالي الرقمي (QTBM Wallet)

تطبيق محفظة رقمية متطور مبني باستخدام Next.js 15 و Firebase، مصمم لتوفير تجربة مالية آمنة وسلسة للمستخدمين في اليمن والمنطقة.

## المميزات الرئيسية

- **إدارة العملات المتعددة**: دعم كامل للريال اليمني (YER)، الدولار الأمريكي (USD)، والريال السعودي (SAR).
- **شراء USDT**: قسم مخصص لشراء العملة الرقمية USDT بعمولة 5% مع معالجة فورية.
- **الأمان والخصوصية**: نظام توثيق هوية صارم (KYC) حيث يتم قفل البيانات القانونية للمستخدم بمجرد حفظها لمنع التلاعب.
- **واجهة مستخدم عصرية**: تصميم يعتمد على Glassmorphism ورسوم متحركة انسيابية مع دعم كامل للوضع الليلي (Dark Mode).
- **نظام دعم متكامل**: محادثة مباشرة بين المستخدم والإدارة لحل الشكاوى وتلقي الاقتراحات.
- **لوحة تحكم إدارية**: إدارة كاملة للمنتجات، أسعار الصرف، طلبات الإيداع، وشكاوي المستخدمين.
- **أوامر صوتية بالذكاء الاصطناعي**: ميزة متقدمة تتيح للمستخدمين تنفيذ العمليات والتنقل عبر التطبيق باستخدام الأوامر الصوتية.

## التقنيات المستخدمة

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS & Shadcn UI
- **Backend**: Firebase (Auth, Firestore)
- **AI**: Genkit for Voice Activated Transactions
- **State Management**: Zustand
- **Icons**: Lucide React

## كيفية البدء

1. قم بتحميل التبعيات:
   ```bash
   npm install
   ```
2. قم بتشغيل بيئة التطوير:
   ```bash
   npm run dev
   ```
3. افتح [http://localhost:9002](http://localhost:9002) في متصفحك.

## تصدير كملف APK
لبناء التطبيق كملف APK لأجهزة الأندرويد، نوصي باستخدام **Capacitor**:
1. قم بإضافة Capacitor للمشروع: `npm i @capacitor/core @capacitor/cli`.
2. قم بإنشاء نسخة الويب: `npm run build`.
3. أضف منصة الأندرويد: `npx cap add android`.
4. قم بمزامنة الملفات: `npx cap sync`.
5. استخدم Android Studio لإنشاء ملف الـ APK النهائي.
