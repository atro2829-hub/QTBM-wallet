
# محفظة QTBM - دليل بناء APK النهائي

لضمان نجاح بناء التطبيق وتجنب الأخطاء التي واجهتها، يرجى اتباع التعليمات التالية بدقة. **يجب دائماً تنفيذ الأوامر من المجلد الرئيسي للمشروع (studio وليس studio/android).**

### الخطوة 1: العودة للمجلد الرئيسي
إذا كنت داخل مجلد android حالياً، اخرج منه أولاً:
```bash
cd ..
```

### الخطوة 2: تشغيل أمر البناء الشامل
لقد قمت بإنشاء أمر واحد يقوم بكل شيء (بناء الموقع، المزامنة، وإنتاج الـ APK):
```bash
npm run apk:full-build
```

---

### في حال استمرار خطأ SDK (SDK Location not found):
1. افتح ملف `android/local.properties`.
2. تأكد من أن مسار `sdk.dir` يطابق مسار الـ Android SDK على جهازك.
   *   إذا كنت تستخدم **Linux** أو بيئة سحابية: عادة ما يكون `/home/user/Android/Sdk`.
   *   إذا كنت تستخدم **Windows**: عادة ما يكون `C:/Users/اسم_المستخدم/AppData/Local/Android/Sdk` (استخدم السلاش الأمامي `/`).

---
تم التطوير بواسطة **فريق QTBM الرقمي**.
