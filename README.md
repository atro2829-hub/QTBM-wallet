
# محفظة QTBM - التميز المالي الرقمي (QTBM Wallet)

تطبيق محفظة رقمية متطور مبني باستخدام Next.js 15 و Firebase، مصمم لتوفير تجربة مالية آمنة وسلسة.

## المميزات الرئيسية
- **إدارة العملات المتعددة**: (YER, USD, SAR) بشكل منفصل تماماً.
- **تحويل فوري بين المستخدمين**: دون الحاجة لموافقة الإدارة.
- **كشف حساب احترافي**: إمكانية استخراج كشف حساب رسمي بصيغة PDF.
- **شراء USDT**: نظام شراء متكامل بعمولة 5%.
- **لوحة تحكم إدارية**: إدارة المستخدمين، المنتجات، والتحويلات.
- **أوامر صوتية بالذكاء الاصطناعي**: تنفيذ العمليات عبر الصوت.

## 🚀 دليل تصدير التطبيق كملف APK (أندرويد)

لتحويل هذا المشروع إلى تطبيق أندرويد أصلي، اتبع الخطوات التالية بدقة:

### 1. المتطلبات الأساسية (على جهازك الكمبيوتر)
- تثبيت **Android Studio**.
- تثبيت **Node.js**.
- تحميل ملفات هذا المشروع إلى جهازك.

### 2. تجهيز المشروع (نفذ هذه الأوامر في الترمنال بالترتيب)
```bash
# 1. بناء نسخة الويب الثابتة
npm run build

# 2. إضافة منصة الأندرويد (تنفذ مرة واحدة فقط)
npx cap add android

# 3. مزامنة ملفات البرمجة مع مجلد الأندرويد
npx cap sync
```

### 3. إنشاء ملف APK النهائي (داخل Android Studio)
1. افتح برنامج **Android Studio**.
2. اختر **Open Project** وقم باختيار مجلد `android` الموجود داخل هذا المشروع.
3. انتظر انتهاء عملية المزامنة (Gradle Sync).
4. من القائمة العلوية اذهب إلى: **Build** > **Build Bundle(s) / APK(s)** > **Build APK(s)**.
5. بمجرد الانتهاء، ستظهر رسالة "APK(s) generated successfully" مع رابط "locate" لفتح المجلد الذي يحتوي على ملف **app-debug.apk**.
